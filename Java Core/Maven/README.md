# Maven POM Full Specification Documentation

This documentation provides a comprehensive and structured reference for all commonly used elements within a Maven `pom.xml`.

## Order of Elements (Recommended Usage Flow)

1. Project Coordinates
   - project
   - modelVersion
   - groupId
   - artifactId
   - version
   - packaging

2. Inheritance & Structure
   - parent
   - modules

3. Dependency Management
   - dependencyManagement
   - dependencies
   - dependency

4. Build Configuration
   - build
   - pluginManagement
   - plugins
   - plugin

5. Environment & Profiles
   - profiles
   - profile

6. Distribution & Repositories
   - repositories
   - distributionManagement

7. Metadata
   - name
   - description
   - url
   - licenses
   - developers
   - scm
